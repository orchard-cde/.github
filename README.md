# orchard-cde/.github

Org-wide defaults for the [orchard-cde](https://github.com/orchard-cde) organization.

## What lives here

- [`profile/README.md`](profile/README.md) — renders on the org's GitHub page at <https://github.com/orchard-cde>.
- `CODE_OF_CONDUCT.md`, `CONTRIBUTING.md`, `SECURITY.md` — inherited by every repo in the org that doesn't ship its own copy.
- `.github/ISSUE_TEMPLATE/`, `.github/PULL_REQUEST_TEMPLATE.md` — inherited issue and PR templates.

See [GitHub's docs on default community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) for inheritance rules.
