# Security Policy

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report security issues privately via [GitHub Security Advisories](https://github.com/orchard-cde/.github/security/advisories/new) on this repository, or on the specific repository where you found the issue.

We aim to acknowledge reports within 3 business days.

## Supported versions

Orchard is pre-1.0. Security fixes are applied to `main` and the latest released minor version on each repo.
